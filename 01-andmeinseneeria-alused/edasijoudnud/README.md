# 01-andmeinseneeria-alused - Edasijõudnud

Selles kaustas asuvad teema "Andmeinseneeria alused" edasijõudnute materjalid.

Kui lood siia uut juhendit, eelda, et õppijal on olemas põhiteadmised käsureast, SQL-ist, Dockerist ja andmebaasidega töötamisest. Hoia keel selge ja rahulik, kuid liigu kiirema tempoga ning keskendu rohkem töövoole, veaotsingule ja tehniliste valikute põhjendamisele.
